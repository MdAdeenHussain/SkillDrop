# SkillDrop

SkillDrop is a mobile-first Flask MVP for blue-collar micro-learning. It includes:

- 60-second lesson flows for electricians, plumbers, drivers, and construction workers
- A simple English and Hindi AI helper backed by mock responses
- Progress tracking, badge placeholders, and job opportunity cards

## Run locally

1. Create and activate a virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Update `.env` if you want to use PostgreSQL instead of the default local SQLite demo database.
4. Start the server with `python app.py`.

The app seeds demo data automatically on first run.
